# 🎮 寝室大亨：28天生存挑战

一个基于 Next.js + TypeScript 开发的校园理财模拟游戏。

![Next.js](https://img.shields.io/badge/Next.js-15-black)
![TypeScript](https://img.shields.io/badge/TypeScript-5-blue)

## 📖 游戏简介

作为一名刚入学的大学生，你需要在 **28 天**内通过投资与兼职，将 **¥500** 变成 **¥2000**。

注意：不能因精力耗尽或欠债而"退学"！

## 🎯 游戏特色

### 💰 多样化的投资系统
- **7只股票**：涵盖内卷、消费、基建、社交四大板块
- **校园混合成长基金**：低风险稳健投资，持有3天后享受复利分红
- **动态价格波动**：每日股价随机波动，市场泡沫可能崩盘
- **财富招忌系统**：资产翻倍触发更多校园突发事件

### 🧠 智力系统
- 智力值影响单股持仓上限
- 学习可提升智力
- 高智力解锁基金投资权限

### ⚡ 行动力限制
- 每天 **2 次行动机会**
- 兼职赚钱 vs 学习提升 vs 市场调研
- 需要精心规划每天的决策

### 📊 板块联动
- 内卷板块上涨 → 社交板块下跌概率增加
- 基建板块受损 → 消费板块剧烈波动
- 特殊事件影响多个相关板块

## 🎮 游戏机制

### 股票交易
- **蜜雪冰城**（社交板块）：受社团招新影响
- **显卡外设**（消费板块）：高波动，受断网/停电事件影响
- **薪火求职**（内卷板块）：受毕业季、面试事件影响
- **考研真题集**（内卷板块）：与智力正相关
- **校园网流量**（基建板块）：低风险
- **共享单车**（基建板块）：受天气事件影响
- **外卖服务**（基建板块）：低风险
- **校园混合成长基金**：混合指数，持有分红

### 随机事件
- 深夜停电 → 显卡暴跌
- 大厂提前批面试 → 薪火求职拉升，学习消耗翻倍
- 极端暴雨 → 共享单车暴跌，外卖上涨
- 室友借钱 → 选择：借钱获得好人卡 vs 拒绝扣精力
- 导师盯上 → 选择：写报告加智力 vs 逃课被强制平仓
- 电脑蓝屏 → 扣维修费，交易锁定

### 每日生存压力
- 扣除 **¥30 生活费**
- 精力耗尽则游戏结束
- 资产归零则破产

## 🛠️ 技术栈

- **框架**: Next.js 15 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **图标**: Lucide React
- **构建工具**: Turbopack

## 📦 安装与运行

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

访问 [http://localhost:3000](http://localhost:3000) 即可开始游戏。

### 构建生产版本
```bash
npm run build
npm start
```

## 🎯 游戏玩法建议

### 初期策略（1-7天）
1. 优先**学习**提升智力，解锁更多持仓上限
2. 兼职赚取启动资金
3. 观察市场波动规律

### 中期策略（8-20天）
1. 智力达标后购买**混合基金**
2. 关注板块联动事件
3. 避开泡沫风险

### 后期策略（21-28天）
1. 稳健持有基金享受分红
2. 关注突发事件风险
3. 精确计算资产达标线

## 🏆 胜利条件

28天后满足以下任一条件：
- ✅ 总资产 ≥ ¥2000：成为寝室大亨
- ❌ 资产归零：破产
- ❌ 精力耗尽：退学
